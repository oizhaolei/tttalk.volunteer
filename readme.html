An openfire plugin. Present web interface: 1. accept translate request, send to some volunteers.
2. accept the "cancel" signal, send to some volunteers.
2. accept the "qa" signal, send to some volunteers.
$ cd $OPENFIRE_HOME/build $ ant plugin -Dplugin=tttalk.volunteer volunteer call example: http://test:9090/plugins/tttalk.volunteer/request?volunteers=volunteer_2002@tttalk.org/volunteer&content=good&fee=23&message_id=458757 http://test:9090/plugins/tttalk.volunteer/cancel?volunteers=volunteer1@tttalk.org/volunteer&message_id=458757
